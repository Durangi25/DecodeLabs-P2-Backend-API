# Student Task API - Backend API Development

## Project Overview

Student Task API is a simple backend REST API developed using Node.js and Express.js as part of the DecodeLabs Full Stack Development Internship Project 2.

The project demonstrates core backend development concepts including API endpoint creation, request and response handling, data validation, JSON processing and HTTP status code management. The API allows users to create and retrieve task information through RESTful endpoints.

## Project Objective

The main objective of this project is to understand backend development fundamentals and learn how applications communicate with servers through APIs.

## Features

* RESTful API architecture
* GET endpoint for retrieving tasks
* GET endpoint for retrieving a specific task
* POST endpoint for creating tasks
* JSON request and response handling
* Input validation
* Proper HTTP status code implementation
* API testing using Postman
* Server-side application logic

## Technologies Used

* Node.js
* Express.js
* JavaScript
* CORS
* Postman

## Testing

The API was tested using Postman.

Tested functionalities:

* API Status Endpoint
* Get All Tasks
* Get Single Task
* Create New Task
* Validation Errors
* HTTP Status Codes
